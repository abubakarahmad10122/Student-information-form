# Student Information Form

## Description

This is a simple Python GUI application built using the Tkinter library. The program allows users to enter their personal information and display it in a large, scrollable output window with a clean and modern dark theme.

## Features

* Enter Full Name
* Enter School
* Enter Class
* Enter Hobbies
* Enter Age
* Display the entered information
* Scrollable output window
* Modern dark-themed user interface
* Simple and beginner-friendly code

## Requirements

* Python 3.x
* Tkinter (included with most Python installations)

## Installation

1. Download or clone this project.
2. Open the project folder.
3. Run the Python file:

```bash
python student_information.py
```

## How to Use

1. Start the application.
2. Enter your:

   * Full Name
   * School
   * Class
   * Hobbies
   * Age
3. Click the **Show Information** button.
4. Your information will appear in the scrollable output window.

## Technologies Used

* Python
* Tkinter

## Theme

* Background Color: Dark Blue
* Input Fields: Slate Blue
* Heading: Golden Yellow
* Output Text: Sky Blue
